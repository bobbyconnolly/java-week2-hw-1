public class Competitor {
    private Pet pet;
    private int score = 0;
    
    public Competitor(Pet pet) {
        this.pet = pet;
    }
    
    public Pet getPet() {
        return pet;
    }
    
    public int getScore() {
        return score;
    }
    
    public void addPoints(int points) {
        this.score += points;
    }
}

Step 2: Create the FetchCompetition

Now let's create our first competition event:

public class FetchCompetition {
    private ArrayList<Competitor> competitors = new ArrayList<>();
    
    public void registerPet(Pet pet) {
        if (canCompete(pet)) {
            competitors.add(new Competitor(pet));
            System.out.println(pet.getName() + " is registered for the fetch competition!");
        } else {
            System.out.println("Sorry, " + pet.getName() + " cannot compete in fetch.");
        }
    }
    
    private boolean canCompete(Pet pet) {
        return pet instanceof Dog;
    }
}

Adding More Events to Our Pet Competition! üé™

Great job on the fetch competition! But a real pet show has many different events. Let's make our competition system more flexible by creating a base class that all competitions can use.

Step 1: Create the Base Competition Class

First, let's create an abstract class that will define what every competition needs:

public abstract class CompetitionEvent {
    private String eventName;
    private ArrayList<Competitor> competitors = new ArrayList<>();
    
    public CompetitionEvent(String eventName) {
        this.eventName = eventName;
    }
    
    public void registerPet(Pet pet) {
        if (canCompete(pet)) {
            competitors.add(new Competitor(pet));
            System.out.println(pet.getName() + " is registered for " + eventName + "!");
        } else {
            System.out.println("Sorry, " + pet.getName() + " cannot compete in " + eventName);
        }
    }
    
    protected ArrayList<Competitor> getCompetitors() {
        return competitors;
    }
    
    public void displayResults() {
        System.out.println("\n=== " + eventName + " Results ===");
        for (Competitor c : competitors) {
            System.out.println(c.getPet().getName() + ": " + c.getScore() + " points");
        }
        
        Competitor winner = findWinner();
        if (winner != null) {
            System.out.println("\nüèÜ Winner: " + winner.getPet().getName() + "! üèÜ");
        }
    }
    
    private Competitor findWinner() {
        if (competitors.isEmpty()) return null;
        
        Competitor winner = competitors.get(0);
        for (Competitor c : competitors) {
            if (c.getScore() > winner.getScore()) {
                winner = c;
            }
        }
        return winner;
    }
    
    // Abstract methods that each competition must implement
    protected abstract boolean canCompete(Pet pet);
    public abstract void runCompetition();
Notice: We've moved the common code (like tracking competitors and displaying results) into the base class. Each specific competition just needs to define who can compete and how the competition runs!

Step 2: Update FetchCompetition

Now let's make FetchCompetition use our new base class:

public class FetchCompetition extends CompetitionEvent {
    public FetchCompetition() {
        super("Fetch Competition");
    }
    
    @Override
    protected boolean canCompete(Pet pet) {
        return pet instanceof Dog;
    }
    
    @Override
    public void runCompetition() {
        for (Competitor comp : getCompetitors()) {
            if (comp.getPet() instanceof Dog) {
                Dog dog = (Dog) comp.getPet();
                // Each dog tries to fetch 3 times
                for (int i = 0; i < 3; i++) {
                    if (dog.knowsFetch()) {
                        comp.addPoints(5);
                    }
                }
            }
        }
    }
}
